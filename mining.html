Mjollnircoin Mining Rewards

Miners are normally awarded with 50 new mjollnircoins per block, an amount which gets halved roughly every 2 years (every 420,000 blocks). The first 4000 blocks have a special reward to encourage miners to start hashing (and thus start securing the network):

  • block 3 - 1000: 125 MNR
  • block 1000 - 2000: 100 MNR
  • block 2000 - 4000: 75 MNR
  • block 4000 - 420000: 50 MNR

Getting Started

  1. Get the Mjollnircoin Wallet
  2. Edit the configuration
  3. Get the cpu-miner or use the built-in miner
  4. Point the miner to your wallet
  5. Start mining

Configuring Mjollnircoin

Edit your mjollnircoin.conf file or create a new one.

The mjollnircoin.conf file is located here:

  1. Linux: $HOME/.mjollnircoin/mjollnircoin.conf
  2. Windows: C:\Users\Username\Appdata\Roaming\Mjollnircoin\mjollnircoin.conf
Edit or add the following lines: 

   rpcpassword=Your_Password_Change_It
   rpcuser=Your_Username_Change_It
   rpcallowip=127.0.0.1
   server=1
If you want to run Mjollnircoin's built-in miner automatically with mjollnircoin-qt, you can add the line 

   gen=1

Using the built-in miner

Start Mjollnircoin-Qt, click Help » Debug Window » Console, then type the following command: 


   setgenerate true 2
This will start two worker threads; You can set the number of worker threads equal to the number of CPU cores you have. To see information about mining, type the following command: 

    getmininginfo

Using the CPU-miner

  • Download the cpu-miner
  • Extract the ZIP file
  • If you are on Windows, open a Command prompt and start the miner

minerd.exe -a heavy -o 127.0.0.1:18320 -u username -p password -t1
  • If you are on Linux, open a Terminal or Shell and start the miner

./minerd -a heavy -o 127.0.0.1:18320 -u username -p password -t1
  • You can set the -t parameter to the number of cpu cores you have
