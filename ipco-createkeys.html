Mjollnircoin's Generate Keys

You can generate a Mjollnircoin address first, using our KeyPair Address Tool. If you are unable to generate a Mjollnircoin address, you can generate a Mjollnircoin address at a later time, after you have downloaded the wallet. The wallet becomes available on {{page.mnr_ipco_mining_launch}}. When the IPO ends, the Mjollnircoins are payed to this Mjollnircoin address. You will be asked for a Mjollnircoin address if we have not received one from you. Once you have downloaded the wallet, you can import your Mjollnircoin Private Key and your balance will be updated with your MNR stake

Escrow service

The escrow for the Mjollnircoin IPO was Escrow.ms, a senior member of the bitcointalk forums.

Generating a Mjollnircoin address

  1. Download the Mjollnircoin Keypair and Address Tool
  2. Unzip the file to your Desktop
  3. Run the tool to generate a Mjollnircoin Private/Public keypair and a Mjollnircoin Address: 
    1) Launch the Command prompt
2) cd %HOME%\Desktop
3) mjollnircoin-genkeys.exe

IPO security

  • We will send you an e-mail or bitmessage confirming your IPO payment has been received.
  • A few days before the release on {{page.mnr_ipco_mining_launch}}, we will publish the list of IPO stakeholders
  • If there are any issues, we will attempt to resolve them before the release on {{page.mnr_ipco_mining_launch}}
  • If you cannot run the KeyPair Address Tool, we can perform the key generation procedure for you. The generated Private and Public Key can be kept on cold storage at your option.
  • We will not share your information with any entity.
